Everything you need to collaborate

Built for developers who want to code together without friction.

⚡
Real-time Collaboration
Multiple users edit the same document simultaneously. Changes appear instantly using Operational Transformation.
🗄️
No Database Needed
Documents live in-memory and expire after 24 hours of inactivity. Zero infrastructure overhead to get started.
🖥️
VS Code–Like Editor
Powered by Monaco Editor — the same engine behind VS Code. Enjoy syntax highlighting, autocomplete, and more.
🐳
Self-Hostable
Deploy on your own server with a single Docker command. Full control over your data and environment.
🌐
50+ Languages
Syntax highlighting for Rust, TypeScript, Python, Go, Java, C++, and many more.
🔗
Instant Sharing
Share a URL — no sign-up, no login, no hassle. Anyone with the link can collaborate immediately.

How It Works

Simple by design, robust under the hood.

1

Open a document

Visit the app and a new unique document URL is generated for you instantly.

2

Share the link

Copy the URL and send it to your collaborators. No accounts or sign-ups required.

3

Code together

Everyone's edits are merged in real time via Operational Transformation — no conflicts, no lost work.

4

Documents auto-expire

Sessions are ephemeral — documents expire after 24 hours of inactivity. No cleanup needed.
